# Overview
The purpose of the SD Standard is to give designers of communications, packaging, and products around the world a tool to help them assess the impact their work has environmentally, socially, culturally and financially. When a project begins, you input the parameters of your project for assessment. (These can be saved as “project” if you have paid a fee.) Your intended targets will be kept in your project file. Once the project is complete, simply confirm that you’ve met the criteria and you will be provided a number that can be posted or printed on the object itself.
