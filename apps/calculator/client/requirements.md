## Packages
framer-motion | Smooth animations for tab transitions, segmented controls, and progress bars
lucide-react | Clean, minimal icons for UI elements
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility to merge Tailwind classes without style conflicts

## Notes
The application relies on a static `/data/criteria.json` file. Ensure this file exists in `client/public/data/` when deployed. No backend API is required; state is managed locally via `localStorage`.
